# HERE AND NOW AI: ML Data for EDA

AI is Good.

This repository contains various datasets specifically curated for Exploratory Data Analysis (EDA) tasks, ranging from housing prices to marketing campaigns and time-series climate data.

## 🚀 Datasets Included

- **Melbourne Housing**: Comprehensive housing data for Melbourne.
- **Preparing for EDA**: Marketing campaign data for concatenation and merging exercises.
- **Visualizing Data**: Housing prices dataset for visualization practice.
- **Univariate/Bivariate/Multivariate Analysis**: Different datasets for statistical analysis.
- **Time-Series Analysis**: Delhi climate and stock data.
- **Text Analysis**: Restaurant reviews for NLP tasks.
