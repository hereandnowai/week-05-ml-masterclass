# Week 05: Machine Learning Masterclass
### AI is Good | https://hereandnowai.com

A comprehensive 5-day Machine Learning curriculum designed for professional bootcamp students. This repository contains hands-on Jupyter Notebooks covering everything from fundamentals to advanced regression models.

## 🚀 Course Structure

- **Day 1: Machine Learning Fundamentals** - AI vs ML vs DL hierarchy, recommendation systems, and decision trees.
- **Day 2: Exploratory Data Analysis (EDA)** - Data cleaning and visualization using real-estate datasets.
- **Day 3: Data Preprocessing** - Building production-grade Scikit-Learn pipelines.
- **Day 4: Linear Regression** - Simple, Multiple, and Polynomial models with evaluation metrics.
- **Day 5: Logistic Regression & Bias-Variance** - Classification, Sigmoid intuition, and model performance analysis.

## 🛠 Tech Stack
- Python
- Scikit-Learn
- Pandas & NumPy
- Matplotlib & Seaborn
