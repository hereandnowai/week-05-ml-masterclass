## "AI is Good"

Welcome to the **Linear Regression Playground**, a modern, interactive web application designed to teach the fundamentals of machine learning through the "Lemonade Stand" analogy. Developed by **HERE AND NOW AI**, this tool helps beginners visualize the relationship between variables and understands how models minimize error.

### 🌟 Key Features
- **Interactive Simulation**: Adjust the **Bias** ($\theta_0$) and **Weight** ($\theta_1$) sliders to see the prediction line move in real-time.
- **Real-time Error Tracking**: Watch the Mean Squared Error (MSE) change as you try to fit the data.
- **Magic Robot Solve**: Use a one-click button to implement the Normal Equation and find the mathematically "perfect" line.
- **Educational Analogies**: Learn using the benefit-driven story of sidewalk money and profit per lemon.

### 🚀 Getting Started
1. Open `index.html` in any modern web browser.
2. Observe the blue dots (your lemonade stand history).
3. Move the sliders to try and find the rule: `Money = Sidewalk Money + (Profit × Lemons)`.
4. Click "Magic Robot Solve" to see how AI handles it!
